# Compiling & launching

All commands must be run from the root project path.

## Server
### Linux y Windows powershell
```bash
./gradlew :server:run
```
### Windows cmd
```bat
.\gradlew :server:run
```

## Client
### Linux y Windows powershell
```bash
./gradlew :client:run
```
### Windows cmd
```bat
.\gradlew.bat :server:run
```
